# Vanusa Azevedo - Psicóloga Landing Page

## Overview
Landing page moderna e profissional para a psicóloga Vanusa Azevedo. O site apresenta informações sobre a profissional, seus serviços, abordagem terapêutica e formas de contato.

## Project Info
- **Profissional**: Vanusa Azevedo - Psicóloga
- **CRP**: 04/635778
- **WhatsApp**: (34) 99228-1523
- **Instagram**: @vanusaa_azevedo

## Tech Stack
- React 18 + Vite
- CSS moderno (variáveis, animações, grid/flexbox)
- Design responsivo

## Project Structure
```
├── src/
│   ├── components/
│   │   ├── Header.jsx/css      # Navegação fixa com menu mobile
│   │   ├── Hero.jsx/css        # Seção principal com CTA
│   │   ├── About.jsx/css       # Sobre a profissional
│   │   ├── Services.jsx/css    # Serviços oferecidos
│   │   ├── Approach.jsx/css    # Processo terapêutico
│   │   ├── Testimonials.jsx/css # Depoimentos
│   │   ├── Contact.jsx/css     # Informações de contato
│   │   ├── Footer.jsx/css      # Rodapé
│   │   └── WhatsAppButton.jsx/css # Botão flutuante WhatsApp
│   ├── App.jsx/css
│   ├── index.css               # Estilos globais
│   └── main.jsx
├── index.html
├── package.json
└── vite.config.js
```

## Features
- Design moderno com paleta de cores sofisticada
- Animações suaves e interações
- Botão flutuante do WhatsApp
- Menu responsivo para mobile
- Seção de depoimentos com slider
- Links diretos para WhatsApp e Instagram

## Running the Project
```bash
npm run dev
```
O servidor roda na porta 5000.
